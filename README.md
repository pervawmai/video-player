# Video Player

This project serves as a practice implementation of a simple video player.

## Features
- Play, pause, and stop functionality
- Volume control
- Fullscreen option

## Tech Stack
- HTML
- CSS
- JavaScript

## Usage
To run this project, clone the repository and open `index.html` in your browser:

```bash
git clone https://github.com/pervawmai/video-player.git
open index.html
```
